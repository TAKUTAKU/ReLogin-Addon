# ReLogin Addon
視聴者様からのご依頼した産物

# 使い方
#### ベータAPIを有効にしてください
初ログインに Tag 「Login」とつく
2回目は Tag 「ReLogin(1)」とつく
3回目以降 () 内の数が増える

初期化する場合Tag 「LoginReset」と付ければ
全てのログインTagが消される

# ダウンロード

PC　右のReleaseから

スマホ　下or上のReleaseから

下の〇〇.mcaddon(.mcpack)っていう青い奴をクリックすると、 ダウンロードできます！
